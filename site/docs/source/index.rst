Create Charts
==================

There are three steps to creating a chart: raw data, wrangling that data, creating the chart.

Add Data
-------------

Easily upload a csv or xls file. Enter the URL of a csv or xls file. Or paste exactly the data you want. 
Upload a table or :sidenote:`Side Note 1!` link to a url.

* Upload from your computer

* Link to URL

* Copy/Paste

:figure:`_static/img/test.png`

Once you have uploaded the file, edit it as necessary.

Modify Data
-------------

Created automated :marginnote:`Margin Note 1.` data modifiers.

* Pivot Tables

* Sort

* New Columns



Create Chart
-------------

Make your chart exactly\ :sidenote:`Side Note 2<img src="_static/img/test.png">` as you like.

* Choose type

* Choose data

* Choose style
